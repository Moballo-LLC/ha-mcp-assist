# HA MCP Assist

MCP-powered Home Assistant conversation agent for efficient entity discovery and tool calling.

- HACS custom integration packaging for `Moballo-LLC/ha-mcp-assist`
- Installs the drop-in `mcp_assist` Home Assistant integration from `custom_components/mcp_assist`
- Uses MCP tools instead of full entity dumps for large Home Assistant systems
- Supports local and cloud LLM providers
- Includes optional calculator, memory, web search, weather, recorder, response-service, and external custom tool families
- Maintains upstream credit to `mike-nott/mcp-assist` while shipping Moballo-maintained releases and support
