Use `search_google_places` when the user asks whether a place is open, where it is, how to call it, or how it is rated. Use `get_google_place_details` when a specific Google Places result needs more detail. Use `get_google_route` for travel time, leave-by, commute, distance, and traffic-aware ETA questions. Only omit `origin` for routes starting from Home Assistant home when home-location tool sharing is enabled; otherwise include a user-provided origin or ask for one. Use `arrival_time` only with transit routes.
